# lab-svip-baijiale

## 项目简介

本仓库用于归档和发布多个独立的 HTML 页面。这些页面以静态文件形式存储，不绑定任何特定域名或网站，仅作为前端资源集合进行管理和展示。

## 目录说明

```
.
├── index.html          # 仓库首页 / 导航页（若有）
├── pages/              # 存放各独立 HTML 页面
│   ├── example1.html
│   ├── example2.html
│   └── ...
├── assets/             # 公共资源（CSS、JS、图片等）
│   ├── css/
│   ├── js/
│   └── images/
└── README.md           # 本文件
```

- `pages/` 目录下每个 HTML 文件均为独立页面，互不依赖。
- `assets/` 目录存放共享资源，供各页面引用（若页面自行包含资源，可忽略此目录）。
- 根目录的 `index.html` 可作为入口导航页（可选）。

## 页面归档说明

- 所有页面均为静态 HTML，可直接在浏览器中打开运行。
- 页面内容可能涉及不同主题或功能，彼此独立。
- 部分页面可能引用外部资源（如 CDN），请确保网络环境允许加载。

## 维护说明

- 新增页面：请在 `pages/` 目录下添加 HTML 文件，并保持命名清晰。
- 更新页面：直接修改对应 HTML 文件，并确保兼容性。
- 删除页面：移除对应文件，并检查是否有相关引用。
- 不建议在页面中嵌入敏感信息或动态后端逻辑。
